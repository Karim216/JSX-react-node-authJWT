Template API REST avec NodeJS et MySQL
====================

Organisation des dossiers et fichiers
-------------------------------------

/api							Code source de l'application
	/node_modules		Dépendances du Projet : Ce dossier inclut toutes les bibliothèques et modules tiers que le projet utilise. Ces dépendances sont
                  définies dans le fichier "package.json" sous la section "dependencies" et "devDependencies".
  /src						Code source principal de l'application
    /auth								Le M de MVC, le coeur de l'application
    /controllers								Le V de MVC, les fichiers statiques (CSS, JS, images, fonts, etc.)
    /database							Feuilles de styles CSS
    /middlewares							Polices de caractères
    /models							Polices de caractères
    /public							Polices de caractères
    /routes							Polices de caractères
    /seedDatabase							Polices de caractères
    middlewares.js		Contrôleur de la page d'accueil
.env								Code source générique, réutilisable dans d'autres projets
package-lock.json					Classe d'accès à la base de données
package.json								Code principal
server.js								Code principal
